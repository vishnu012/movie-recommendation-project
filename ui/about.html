FLIXGO – Best Place for Recommend Movies

Our project belongs to the creation of a Web Application.A user search a movie name he already seen. We also provide Ratings and other informations of suggested movies from various sites like Rotten Tomatoes ,imdb etc..

Our project aims to provide a content based movie recommendation system.Users can find similar movies instantly.
